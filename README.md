Установка Visual Studio:
зайдите на сайт visualstudio.microsoft.com и скачайте бесплатную версию Community;
установите Visual Studio с компонентами для разработки Windows Forms.
Скачивание и открытие проекта:
распакуйте архив и откройте файл решения (.sln) в Visual Studio.
Настройка среды:
проверьте наличие файла products.json в каталоге проекта — он должен содержать список товаров в формате JSON;
можно создать файл вручную или сгенерировать через интерфейс.
Запуск проекта:
нажмите кнопку "Start" или клавишу F5 для запуска программы в режиме отладки.
Работа с приложением:
в интерфейсе выберите товары, добавляйте их в корзину, оформляйте заказ;
для тестирования зарегистрируйте нового пользователя через форму регистрации (при необходимости доработайте интерфейс).
